# annotationRuntime
运行时注释，范例；

使用 annotation RunTime 实现 BufferKnifer类似功能； 
Annotation 可以实现一些简单的初始化，让Code更简单；
### 已完成
* @JFindView(<ViewId>)
支持 findViewById 
* @JFindViewOnClick
支持 findViewById and  onClick
> Activity 需要 implement View.onClick.class;
* @JIntent(<key>)
支持Intent.getStringExtra(<String>)
